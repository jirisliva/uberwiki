## How to Create New File
1. Edit any existing file.
2. Create link to file with wanted name. E.g.:

    `[New File](new file.md)`

3. Save file.
4. Click to link.

## Markdown Formatting

### Bullet List

### Blockquotes
> Just put `>` as first character of text which should be quoted.

    public void static main() {

    }
